## Project Description

This is a simple and responsive To-Do application built using **React JS**.  
The app uses the **useReducer hook** to manage all task operations including adding, updating, deleting, and completing tasks.

The project also uses **localStorage** to save tasks so they remain even after refreshing the page.


## Features

- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as completed
- Tasks stay in same position after update
- Data persistence using localStorage
- Interactive icons using Font Awesome
- Responsive and clean UI design


## Technologies Used

- React JS
- useReducer Hook
- JavaScript (ES6)
- HTML5
- CSS3
- Font Awesome Icons
- localStorage
